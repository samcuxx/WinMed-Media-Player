## Installation

```bash
npm install
```

## Usage

Start the application:

```bash
npm start
```

Development mode:

```bash
npm run dev
```

Build for production:

```bash
npm run package
```

## Keyboard Shortcuts

| Action                | Shortcut           |
| --------------------- | ------------------ |
| Play/Pause            | `Space`            |
| Seek Forward/Backward | `Right/Left Arrow` |
| Volume Up/Down        | `Up/Down Arrow`    |
| Toggle Fullscreen     | `F`                |
| Toggle Mute           | `M`                |
| Toggle PiP            | `Alt + P`          |
